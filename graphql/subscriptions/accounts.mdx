Subscribe to the accounts updates

```graphql
accounts(
  marketId: ID = 0
  partyId: ID = 0
  asset: String
  type: AccountType
): Account!

```

### Arguments

#### `marketId` ([`ID`](/graphql/scalars/id))

ID of the market from which we want accounts updates

#### `partyId` ([`ID`](/graphql/scalars/id))

ID of the party from which we want accounts updates

#### `asset` ([`String`](/graphql/scalars/string))

Asset code

#### `type` ([`AccountType`](/graphql/enums/account-type))

Type of the account

### Type

#### [`Account`](/graphql/objects/account)

An account record
