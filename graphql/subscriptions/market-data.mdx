Subscribe to the mark price changes

```graphql
marketData(
  marketId: ID = 0
): MarketData!

```

### Arguments

#### `marketId` ([`ID`](/graphql/scalars/id))

id of the market we want to subscribe to the market data changes

### Type

#### [`MarketData`](/graphql/objects/market-data)

Live data of a Market
