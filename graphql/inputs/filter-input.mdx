title: FilterInput
---

Filter describes the conditions under which an oracle data is considered of
interest or not.

```graphql
type FilterInput {
  key: PropertyKeyInput!
  conditions: [ConditionInput!]
}
```

### Fields

#### `key` ([`PropertyKeyInput!`](/graphql/inputs/property-key-input))

key is the oracle data property key targeted by the filter.

#### `conditions` ([`[ConditionInput!]`](/graphql/inputs/condition-input))

conditions are the conditions that should be matched by the data to be
considered of interest.
