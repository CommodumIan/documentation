PriceMonitoringParameters holds together price projection horizon Ï„, probability level p, and auction extension duration

```graphql
type PriceMonitoringTriggerInput {
  horizonSecs: Int!
  probability: Float!
  auctionExtensionSecs: Int!
}
```

### Fields

#### `horizonSecs` ([`Int!`](/graphql/scalars/int))

Price monitoring projection horizon Ï„ in seconds (> 0).

#### `probability` ([`Float!`](/graphql/scalars/float))

Price monitoring probability level p. (>0 and < 1)

#### `auctionExtensionSecs` ([`Int!`](/graphql/scalars/int))

Price monitoring auction extension duration in seconds should the price
breach it's theoretical level over the specified horizon at the specified
probability level (> 0)
