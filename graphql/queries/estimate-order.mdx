id: estimate-order
title: estimateOrder
---

return an estimation of the potential cost for a new order

```graphql
estimateOrder(
  marketId: ID!
  partyId: ID!
  price: String
  size: String!
  side: Side!
  timeInForce: OrderTimeInForce!
  expiration: String
  type: OrderType!
): OrderEstimate!

```

### Arguments

#### `marketId` ([`ID!`](/graphql/scalars/id))

ID of the market to place the order

#### `partyId` ([`ID!`](/graphql/scalars/id))

ID of the party placing the order

#### `price` ([`String`](/graphql/scalars/string))

Price of the asset

#### `size` ([`String!`](/graphql/scalars/string))

Size of the order

#### `side` ([`Side!`](/graphql/enums/side))

Side of the order (Buy or Sell)

#### `timeInForce` ([`OrderTimeInForce!`](/graphql/enums/order-time-in-force))

TimeInForce of the order

#### `expiration` ([`String`](/graphql/scalars/string))

expiration of the the order

#### `type` ([`OrderType!`](/graphql/enums/order-type))

type of the order

### Type

#### [`OrderEstimate`](/graphql/objects/order-estimate)

An estimate of the fee to be paid by the order
