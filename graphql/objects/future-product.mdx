No description

```graphql
type FutureProduct {
  maturity: String!
  settlementAsset: Asset!
  quoteName: String!
  oracleSpec: OracleSpecConfiguration!
  oracleSpecBinding: OracleSpecToFutureBinding!
}
```

### Fields

#### `maturity` ([`String!`](/graphql/scalars/string))

RFC3339Nano time when the future products matures

#### `settlementAsset` ([`Asset!`](/graphql/objects/asset))

Product asset ID

#### `quoteName` ([`String!`](/graphql/scalars/string))

String representing the quote (e.g. BTCUSD -> USD is quote)

#### `oracleSpec` ([`OracleSpecConfiguration!`](/graphql/objects/oracle-spec-configuration))

Describes the oracle data that an instrument wants to get from the oracle engine.

#### `oracleSpecBinding` ([`OracleSpecToFutureBinding!`](/graphql/objects/oracle-spec-to-future-binding))

OracleSpecToFutureBinding tells on which property oracle data should be
used as settlement price.
