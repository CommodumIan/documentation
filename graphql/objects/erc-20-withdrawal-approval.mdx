title: Erc20WithdrawalApproval
---

All the data related to the approval of a withdrawal from the network

```graphql
type Erc20WithdrawalApproval {
  assetSource: String!
  amount: String!
  expiry: String!
  nonce: String!
  signatures: String!
}
```

### Fields

#### `assetSource` ([`String!`](/graphql/scalars/string))

The source asset in the ethereum network

#### `amount` ([`String!`](/graphql/scalars/string))

The amount to be withdrawn

#### `expiry` ([`String!`](/graphql/scalars/string))

Timestamp in seconds for expiry of the approval

#### `nonce` ([`String!`](/graphql/scalars/string))

The nonce to be used in the request

#### `signatures` ([`String!`](/graphql/scalars/string))

Signature aggregate from the nodes, in the following format:
0x + sig1 + sig2 + ... + sigN
