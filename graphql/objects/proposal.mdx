No description

```graphql
type Proposal {
  id: ID
  reference: String!
  party: Party!
  state: ProposalState!
  datetime: String!
  terms: ProposalTerms!
  votes: ProposalVotes!
  rejectionReason: ProposalRejectionReason
}
```

### Fields

#### `id` ([`ID`](/graphql/scalars/id))

Proposal ID that is filled by VEGA once proposal reaches the network

#### `reference` ([`String!`](/graphql/scalars/string))

A UUID reference to aid tracking proposals on VEGA

#### `party` ([`Party!`](/graphql/objects/party))

Party that prepared the proposal

#### `state` ([`ProposalState!`](/graphql/enums/proposal-state))

State of the proposal

#### `datetime` ([`String!`](/graphql/scalars/string))

RFC3339Nano time and date when the proposal reached Vega network

#### `terms` ([`ProposalTerms!`](/graphql/objects/proposal-terms))

Terms of the proposal

#### `votes` ([`ProposalVotes!`](/graphql/objects/proposal-votes))

Votes cast for this proposal

#### `rejectionReason` ([`ProposalRejectionReason`](/graphql/enums/proposal-rejection-reason))

Reason for the proposal to be rejected by the core
