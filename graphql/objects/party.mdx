Represents a party on Vega, could be an ethereum wallet address in the future

```graphql
type Party {
  id: ID!
  orders(skip: Int = 0, first: Int = 0, last: Int = 0): [Order!]
  trades(
    marketId: ID = 0
    skip: Int = 0
    first: Int = 0
    last: Int = 0
  ): [Trade!]
  accounts(marketId: ID = 0, asset: String, type: AccountType): [Account!]
  positions: [Position!]
  margins(marketId: ID = 0): [MarginLevels!]
  proposals(inState: ProposalState): [Proposal]
  votes: [ProposalVote]
  withdrawals: [Withdrawal!]
  deposits: [Deposit!]
  liquidityProvisions(market: String, reference: String): [LiquidityProvision!]
}
```

### Fields

- ##### `marketId` ([`ID`](/graphql/scalars/id))

- ##### `last` ([`Int`](/graphql/scalars/int))

#### `accounts` ([`[Account!]`](/graphql/objects/account))

- ##### `asset` ([`String`](/graphql/scalars/string))

Asset (USD, EUR etc)
