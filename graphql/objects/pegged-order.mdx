Create an order linked to an index rather than a price

```graphql
type PeggedOrder {
  reference: PeggedReference!
  offset: String!
}
```

### Fields

#### `reference` ([`PeggedReference!`](/graphql/enums/pegged-reference))

Index to link this order to

#### `offset` ([`String!`](/graphql/scalars/string))

Price offset from the peg
