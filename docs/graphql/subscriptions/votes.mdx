Subscribe to votes, either by proposal id or pary id

```graphql
votes(
  proposalId: ID = 0
  partyId: ID = 0
): ProposalVote!

```

### Arguments

#### `proposalId` ([`ID`](/docs/graphql/scalars/id))

Optional proposal id which votes are to be streamed

#### `partyId` ([`ID`](/docs/graphql/scalars/id))

Optional party id whose votes are to be streamed

### Type

#### [`ProposalVote`](/docs/graphql/objects/proposal-vote)
