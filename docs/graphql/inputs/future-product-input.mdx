Future product configuration

```graphql
type FutureProductInput {
  maturity: String!
  settlementAsset: String!
  quoteName: String!
  oracleSpec: OracleSpecConfigurationInput!
  oracleSpecBinding: OracleSpecToFutureBindingInput!
}
```

### Fields

#### `maturity` ([`String!`](/docs/graphql/scalars/string))

Future product maturity (ISO8601/RFC3339 timestamp)

#### `settlementAsset` ([`String!`](/docs/graphql/scalars/string))

Product asset name

#### `quoteName` ([`String!`](/docs/graphql/scalars/string))

String representing the quote (e.g. BTCUSD -> USD is quote)

#### `oracleSpec` ([`OracleSpecConfigurationInput!`](/docs/graphql/inputs/oracle-spec-configuration-input))

The oracle spec describing the oracle data of interest.

#### `oracleSpecBinding` ([`OracleSpecToFutureBindingInput!`](/docs/graphql/inputs/oracle-spec-to-future-binding-input))

The binding between the oracle spec and the settlement price
