title: prepareOrderAmend
---

Send an amend order request to be prepared. Returns a pending order + blob of the transaction to submit.
The data is verified. The response can be signed and submitted through the submitTransaction mutation.

```graphql
prepareOrderAmend(
  id: ID!
  price: String!
  sizeDelta: String!
  expiration: String
  timeInForce: OrderTimeInForce!
  peggedReference: PeggedReference
  peggedOffset: String
): PreparedAmendOrder!

```

### Arguments

#### `id` ([`ID!`](/docs/graphql/scalars/id))

ID of the order to amend

#### `price` ([`String!`](/docs/graphql/scalars/string))

New price for this order

#### `sizeDelta` ([`String!`](/docs/graphql/scalars/string))

New size for this order

#### `expiration` ([`String`](/docs/graphql/scalars/string))

New expiration time

#### `timeInForce` ([`OrderTimeInForce!`](/docs/graphql/enums/order-time-in-force))

Time in force

#### `peggedReference` ([`PeggedReference`](/docs/graphql/enums/pegged-reference))

The reference point for pegged orders

#### `peggedOffset` ([`String`](/docs/graphql/scalars/string))

The offset from the reference price for pegged orders

### Type

#### [`PreparedAmendOrder`](/docs/graphql/objects/prepared-amend-order)
