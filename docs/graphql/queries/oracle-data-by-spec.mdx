All oracle data for a given oracle spec ID

```graphql
oracleDataBySpec(
  oracleSpecID: String!
): [OracleData!]

```

### Arguments

#### `oracleSpecID` ([`String!`](/docs/graphql/scalars/string))

ID for an oracle spec

### Type

#### [`OracleData`](/docs/graphql/objects/oracle-data)

An oracle data contains the data sent by an oracle
