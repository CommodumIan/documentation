Represents an asset in vega

```graphql
type Asset {
  id: ID!
  name: String!
  symbol: String!
  totalSupply: String!
  decimals: Int!
  minLpStake: String!
  source: AssetSource!
  infrastructureFeeAccount: Account!
}
```

### Fields

#### `id` ([`ID!`](/docs/graphql/scalars/id))

The id of the asset

#### `name` ([`String!`](/docs/graphql/scalars/string))

The full name of the asset (e.g: Great British Pound)

#### `symbol` ([`String!`](/docs/graphql/scalars/string))

The symbol of the asset (e.g: GBP)

#### `totalSupply` ([`String!`](/docs/graphql/scalars/string))

The total supply of the market

#### `decimals` ([`Int!`](/docs/graphql/scalars/int))

The precision of the asset

#### `minLpStake` ([`String!`](/docs/graphql/scalars/string))

The min stake to become an lp for any market using this asset for settlement

#### `source` ([`AssetSource!`](/docs/graphql/unions/asset-source))

The origin source of the asset (e.g: an erc20 asset)

#### `infrastructureFeeAccount` ([`Account!`](/docs/graphql/objects/account))

The infrastructure fee account for this asset
