A Future product

```graphql
type Future {
  maturity: String!
  settlementAsset: Asset!
  quoteName: String!
  oracleSpec: OracleSpec!
  oracleSpecBinding: OracleSpecToFutureBinding!
}
```

### Fields

#### `maturity` ([`String!`](/docs/graphql/scalars/string))

RFC3339Nano maturity date of the product

#### `settlementAsset` ([`Asset!`](/docs/graphql/objects/asset))

The name of the asset (string)

#### `quoteName` ([`String!`](/docs/graphql/scalars/string))

String representing the quote (e.g. BTCUSD -> USD is quote)

#### `oracleSpec` ([`OracleSpec!`](/docs/graphql/objects/oracle-spec))

The oracle spec describing the oracle data of interest.

#### `oracleSpecBinding` ([`OracleSpecToFutureBinding!`](/docs/graphql/objects/oracle-spec-to-future-binding))

The binding between the oracle spec and the settlement price
